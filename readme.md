## 使用方法
### 一、使用exe文件
1. 下载并运行exe程序

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/26061940/1771772814947-7aa4b424-d0dc-424b-b9e8-a7761b6984d8.png)

程序运行后，会在同目录下自动生成config文件夹，用于存储用户的个性化设置。

2. 导入Excel花名册

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/26061940/1771772914715-6a16e80e-1ae8-4fb4-b5b6-8a1817b32302.png)

导入后，系统会自动识别Excel表格首行的标题，并将其作为格式配置的参考变量。

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/26061940/1771773207768-bb36bf90-463d-45cc-926b-e351ba2cb491.png)

在「管理格式」模块中，可新建自定义的文件命名格式，格式中可插入的变量均来源于导入Excel表格的首行标题。

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/26061940/1771773355771-f09e9b96-899e-4054-a923-923374837dde.png)

3. 扫描文件并生成结果  
选择待扫描的文件夹和扫描结果的输出文件夹，点击「开始扫描」即可获取以下结果：

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/26061940/1771773525925-cf13cdbc-9a17-4921-8028-76ac875a46db.png)

+ 未提交作业的学生名单；
+ 重复提交作业的学生名单；
+ 自动重命名不符合格式要求的文件（文件名称中需包含「班级」「姓名」「学号」任一元素）。

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/26061940/1771773701107-51b4782c-f70f-48cf-8f35-05f06bd6eb71.png)

如上图所示，学生上传的文件会按照预设格式完成重命名。

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/26061940/1771773891074-2473e3c2-798d-42e6-ac38-269053d5216c.png)

4. 批量处理作业  
选择母文件夹后，勾选需要处理的子文件夹，即可完成批量操作：

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/26061940/1771774020386-09740d49-0944-4bea-99f3-b325ffaf920b.png)

批量处理完成后，系统会生成作业提交情况汇总报告，其中红色单元格代表对应学生未提交作业。

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/26061940/1771774090745-485d0198-e92c-4e78-911b-b3e4f663c8af.png)

## 后记
收取作业是一项低技术含量但重复性极高的工作，想要让每位学生都严格按照要求提交作业往往难以实现。为此，我开发了这款工具来摆脱此类重复性劳动，也希望它能为更多有需要的人提供帮助。
